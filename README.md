## üîó Setting Up Discord Notifications

To receive notifications on Discord:

1. Create or choose a Discord server where you want to receive alerts.

2. Go to the server settings and find the "Bots" or "Integrations" section.

3. Create a new bot and obtain the bot token.

4. Open the tech_news application and enter your Discord bot token in the designated field.

5. Choose the channels where you want to receive security updates.

6. Save your settings.

Now, the tech_news application will send security alerts directly to your Discord channels.
